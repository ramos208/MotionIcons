# BanknoteX Icon

Dynamic dynamic animated asset part of the Universal Motion Engine powered by Ollama AI.

## AI Philosophy
> The icon is designed to mimic the movement of a banknote, starting with an elastic-pop effect that simulates the bill's springiness. Upon hovering, subtle particle effects add a dynamic glow, and upon clicking, a slight vibration ensures user interaction feedback.
