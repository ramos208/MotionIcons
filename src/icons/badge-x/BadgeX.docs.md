# BadgeX Icon

Dynamic dynamic animated asset part of the Universal Motion Engine powered by Ollama AI.

## AI Philosophy
> The icon badge-x adopts an elastic-pop motion type to highlight its key features while maintaining a visually engaging and interactive experience. The idle animation emphasizes the badge's presence with subtle rotation and scale adjustments, creating a sense of depth and interactivity. Upon hovering, the badge experiences a spring-like expansion that mimics the pop effect of badges, making it stand out more prominently. When interacted with, a gentle vibration is triggered to enhance user experience without being overwhelming.
