## AI Philosophy
> The motion profile for the asterisk-square icon is designed to be visually engaging and semantically accurate, using an elastic-pop approach that simulates a spring-like behavior typical of icons with directional movement such as chevrons. The idle animation introduces a subtle rotation and scale variation, while hover triggers a more pronounced visual effect like a light flash. Upon click, a ripple effect is generated to enhance the interactive nature.
