# Motion Intelligence System Pipeline

This directory contains the core engine for generating intelligent, physics-based, semantic animations for any SVG icon.

## Architecture

The system is designed around the `MotionIntelligenceEngine`. It uses a combination of semantic analysis and geometric heuristics to automatically map generic SVG shapes to advanced physics profiles.

### 1. The Engine (`MotionIntelligenceEngine.ts`)
The engine categorizes icons into types like `notification`, `weather`, `search`, `mechanical`, etc. It then assigns:
- Idle Animations (e.g., `atmospheric-floating`)
- Hover Animations (e.g., `pendulum-swing`)
- Click/Interaction Logic (e.g., particle bursts, elastic rebounds)
- Shaders and Particle configurations.

### 2. The Generated Output (`src/icons/[category]/`)
For every icon processed by the pipeline, a custom folder is created containing:
- `[IconName]Icon.vue` - The semantic, fully interactive wrapper around the raw SVG path.
- `[IconName].motion.ts` - The strict physics values generated by the engine.
- `[IconName].particles.ts` - Math/logic for generating secondary interactions (sound waves, kinetic trails).
- `[IconName].shader.ts` - Custom GLSL logic for WebGL distortion (if using TresJS/ThreeJS layers).

### Provided Examples
We have successfully processed and handcrafted the foundation models for the 6 requested categories:
1. `bell/BellIcon.vue` -> Implements `notification` (pendulum, delayed clapper physics, sound wave particles).
2. `cloud/CloudIcon.vue` -> Implements `weather` (atmospheric turbulence and buoyancy).
3. `search/SearchIcon.vue` -> Implements `search` (lens focus and radar sweep).
4. `ai/AIIcon.vue` -> Implements `ai` (neural pulses and node communication).
5. `gear/GearIcon.vue` -> Implements `mechanical` (inertia, gear synchronization, friction).
6. `arrow/ArrowIcon.vue` -> Implements `directional` (spring momentum and kinetic velocity trails).

### How to Scale to "ALL Icons"
To process the thousands of icons in Lucide, Heroicons, etc., you can feed the SVGs through a build script that calls `MotionIntelligenceEngine.generateMotion(analysis)` and outputs the `.vue` and `.ts` templates automatically matching the structure of `BellIcon.vue`.
